# FleetInfo

## Setup
The project is powered via API keys, to succesfuly compile the application you will need to add the following in your environment
variables:
ACCOUNT_TOKEN=[YOUR TOKEN]
AUTHORIZATION_TOKEN=[AUTHORIZATION TOKEN]


## Tech Stack
- MVVM - UseCases
- Compose
- Coroutines and Flow
- Coil Image loading
- Hilt
- Paging3
- Retrofit2 / OkHttp
